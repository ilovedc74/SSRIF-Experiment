## Socially-Shared Retrieval-Induced Forgetting Experiment

### Introduction

This is a webpage created to conduct Retrieval-induced forgetting paradigm experiment.

There are 5 stages.

 #1 Memory stage
 #2 film QA
 #3 Story Discussion
 #4 Arithmetic
 #5 Recall

Pilot studies experiment 1 and 2 are not included.
